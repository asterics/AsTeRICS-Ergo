Help - FlipMouse

General Information

The FlipMouse is an alternative computer mouse (and keyboard), which makes it possible to control a computer e.g. by mouth or minimal finger movement.

In addition the FlipMouse is capable of learning commands from infrared remote controls. Thus it is possible to replace remote controls of various devices with AsTeRICS Ergo.

Get a FlipMouse

Like AsTeRICS Ergo the FlipMouse was developed in several research projects and is now maintained by the AsTeRICS Foundation. Since it is a non-commercial open source product, these are the only possibilities to get a FlipMouse:

  • Order a FlipMouse DIY kit at the Hackerspace Webshop
  • Request via e-mail for persons which cannot afford or cannot build the DIY kit.
