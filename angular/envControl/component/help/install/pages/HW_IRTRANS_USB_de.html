Installation von IrTrans-USB

Wenn Sie bereits einen IrTrans-USB besitzen, können Sie diesen wie folgt installieren:

  1. Verbinden Sie den IrTrans mit einem USB-Kabel mit dem Computer (USB Typ A-B, siehe {{'i18n_ec_help' | translate}}).
  2. Installieren Sie den benötigten Treiber für Windows, indem Sie auf den Button "Treiber installieren" klicken. Dabei wird es 2 Sicherheitsabfragen geben, die Sie mit "Ja" bestätigen müssen.
    Treiber installiert!

    Hinweis: Möglicherweise öffnen sich die Fenster für die Treiberinstallation im Hintergrund und müssen von der Taskleiste geöffnet werden:
  3. Warten Sie 10 Sekunden auf die Fertigstellung der Hardwareinstallation.
  4. Klicken Sie auf "Server Starten" um ein für den IrTrans benötigte Programm im Hintergrund zu starten. Nach dieser Installation wird es von AsTeRICS Ergo bei Bedarf automatisch gestartet.
    Server gestartet!
  5. Testen der Installation - Klicken Sie auf "Testen" um die korrekte Installation zu prüfen.
