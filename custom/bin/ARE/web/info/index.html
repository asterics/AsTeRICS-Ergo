Allgemeine Informationen

AsTeRICS-Ergo ist eine freie Open Source Software, die es ermöglicht Unterstützungstechnologien einfach und kostengünstig einzurichten. Es ist eine Erweiterung des AsTeRICS-Frameworks mit einer intuitiven und flexiblen Web-Benutzeroberfläche.

Anwendungsgebiete für AsTeRICS-Ergo:

  • Umgebungssteuerung
  • Alternative Computersteuerungen wie Mundmaus, Kopfmaus oder Buttons (in Planung)
