Da AsTeRICS im Rahmen von Forschungsprojekten entstanden ist, gibt es keinen offiziellen technischen Support.

AsTeRICS Ergo wird aber von einem gemeinnützigen Verein, der AsTeRICS-Foundation, weiterentwickelt und betreut. Bei Fragen oder Problemen gibt es daher folgende Möglichkeiten:

  1. E-Mail an info@asterics-foundation.org
  2. Fehler oder technische Probleme können auch direkt auf github.com gemeldet bzw. eingetragen werden.
