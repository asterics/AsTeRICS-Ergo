Zum Löschen bereits eingerichteter Befehle oder Geräte haben Sie 2 Möglichkeiten:
  1. Klick auf den Button „Löschen aktivieren“: Der nächste Button, der ausgewählt wird, wird gelöscht.
  2. Klicken Sie mit der rechten Maustaste auf den Button, den Sie löschen möchten. Anschließend öffnet sich ein Menü, in welchem Sie „Löschen“ auswählen können.
