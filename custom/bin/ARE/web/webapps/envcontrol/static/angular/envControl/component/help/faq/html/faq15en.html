There are 3 possibilities:
  1. Check whether there is intervisibility between the IrTrans and the device (which you wish to control).
  2. Delete the commands, which do not work. Add them again using the button „new command“. Caution: It is important that the remote control points in the direction of the IrTrans (not the device) while setting up the control.
  3. Delete the device and restart the setup.
You can delete devices or commands using the button “Activate Deleting” or via right-hand click.
