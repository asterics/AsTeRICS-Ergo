Like most websites AsTeRICS Ergo can be used with the keyboard. There are the following possibilities for navigation:

  • ↹ (Tab):
    Moves to the next selectable element on the page. This can be a link, a button or a combobox. The currently active element is shown by a colored border.
  • ⇧ + ↹ (Shift + Tab):
    Moves to the previous selectable element on the page. This can be again a link, a button or a combobox.
  • Space oder Enter
    Selects the active element. Presses the active button or follows the active link.
  • Alt + ◀ (Alt + Arrow left)
    Back to the previous page.
