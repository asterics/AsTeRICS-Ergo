Ja, Sie können mit AsTeRICS Ergo auch Geräte steuern, zu denen es noch keinen Assistenten gibt. Gehen Sie hierfür auf Gerät hinzufügen –> Weitere…. Wählen Sie anschließend, ob Sie ein Fernbedienungs-Gerät (mit dem IrTrans) oder ein Gerät über die Steckdose (mit dem FS20-System) steuern möchten und folgen Sie den weiteren Anweisungen.

Wählen Sie „Steckdose“, benötigen Sie das FS20-System. Damit können Geräte, die an einer Steckdose hängen, ein- und ausgeschalten werden. Beispiel: Ventilator

Wählen Sie „Fernbedienungs-Gerät“, benötigen Sie das IrTrans-System. Damit ist es möglich beinahe alle gängigen Fernbedienungen (Infrarot) zu ersetzen. Beispiele für weitere Steuerungsmöglichkeiten: Fenster, Jalousien, Türen, Staubsaugerroboter, …
