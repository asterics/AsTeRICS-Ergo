On Windows-Tablets it is possible to install AsTeRICS-Ergo directly, like on any PC or Laptop.

If you want to use AsTeRICS-Ergo on an other Tablet or Smartphone follow these steps:

  1. AsTeRICS-Ergo has to be installed and started on a computer
  2. Connect the computer and the Smartphone/Tablet with the same wireless network (wifi).
  3. On the Smartphone/Tablet enter the following address in the address bar:
    {{$ctrl.getLocalAsTeRICSAddress()}}
  4. Alternatively you can scan the following QR-code:
  5. On the Smartphone/Tablet enter the following address in the address bar:
    <IP-address of the computer>:{{$ctrl.getLocalPort()}}/webapps/envcontrol/static/
