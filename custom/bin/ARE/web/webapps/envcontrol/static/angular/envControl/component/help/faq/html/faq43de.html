AsTeRICS Ergo kann wie die meisten Webseiten mit der Tastatur verwendet werden. Dabei gibt es folgende Möglichkeiten zur Navigation:

  • ↹ (Tabulator):
    Springt zum nächsten auswählbaren Element auf der Seite. Das kann ein Link, ein Button oder eine Auswahlliste sein. Das aktuell ausgewählte Element ist durch eine farbige Umrahmung ersichtlich.
  • ⇧ + ↹ (Umschalttaste + Tabulator):
    Springt zum vorherigen auswählbaren Element auf der Seite. Das kann ebenfalls ein Link, ein Button oder eine Auswahlliste sein.
  • Leertaste oder Enter
    Wählt das aktuelle Element aus. Drückt den aktuell ausgewählten Button, oder folgt dem aktuell ausgewählten Link.
  • Alt + ◀ (Alt + Pfeiltaste nach links)
    Zurück zur letzten Seite.
