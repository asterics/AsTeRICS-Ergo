Installation of the IrTrans USB

If you have already purchased an IrTrans USB, you can follow the instructions to install it:

  1. Connect the IrTrans to the computer using an USB cable (type A-B, see {{'i18n_ec_help' | translate}}).
  2. Install the needed driver for Windows by clicking on the button "Install Driver". There will be 2 confirmation prompts that have to be confirmed with "Yes".
    driver installed!

    Hint: Maybe the windows for driver installation do not pop up but have to be opened from the taskbar:
  3. Wait 10 seconds in order to wait for completion of the hardware installation.
  4. Click on "Start Server" in order to start the service program needed for IrTrans. After this installation it will be started automatically by AsTeRICS Ergo, if needed.
    server started!
  5. Test the installation - Select "Test" to check whether the IrTrans has been installed successfully.
