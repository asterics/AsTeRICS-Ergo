Hilfe - Infrarot Stecker

Allgemeine Informationen

Infrarot Stecker können über ein Infrarot Signal einer Fernbedienung ein- bzw. ausgeschaltet werden. Dadurch ist die Steuerung einer Steckdose in Kombination mit einem IrTrans USB oder der FLipMouse möglich. So kann beispielsweise eine Tischlampe oder ein Ventilator ein- und ausgeschaltet werden.
