Es kann nicht verallgemeinert werden, welche zusätzliche Hardware erworben werden muss. Dies hängt davon ab, welche Geräte mit AsTeRICS Ergo gesteuert werden sollen. Verwenden Sie den Bestell-Assistenten um herauszufinden, welche zusätzliche Hardware gekauft werden muss. Wählen sie alle Geräte, die gesteuert werden sollen, aus. Die Hardware, die Sie zur Steuerung benötigen, wird darunter angezeigt.
  1. Möchten Sie ein Gerät steuern, das normalerweise per Fernbedienung bedient wird, benötigen Sie für die Steuerung einen IrTrans (TODO Link Hilfe IrTrans). Dieser ermöglicht es Fernbedienungsbefehle zu ersetzen. Es muss immer nur ein IrTrans erworben werden, da dieser mehrere Fernbedienungsgeräte steuern kann. Außer dem IrTrans ist für die Steuerung dieser Geräte keine weitere zusätzliche Hardware notwendig. Achtung, bei der Bestellung wird kein Kabel mitgeliefert. Aus diesem Grund muss zusätzlich ein USB Typ B Kabel bestellt werden.
  2. Möchten Sie das Raumlicht via AsTeRICS Ergo steuern, ist es notwendig die herkömmlichen Glühbirnen mit IR-Glühbirnen (TODO Link Hilfe IR-Glühbirne) auszutauschen. Diese besitzen eine Fernbedienung und können anschließend mit dem IrTrans gesteuert werden.
  3. Möchten Sie ein Gerät steuern, das an der Steckdose angeschlossen ist und normalerweise mit einem Schalter ein- und ausgeschalten wird (z.B. Stehlampe, Ventilator), benötigen Sie das FS20-System (TODO Link Hilfe FS20). Über das FS20-System können Geräte ein- und ausgeschaltet werden, jedoch keine stufenweise Änderungen vorgenommen werden. Beispiel: Eine Stehlampe kann mit dem FS20-System ein- und ausgeschaltet werden, die einzelnen Helligkeitsstufen können jedoch nicht verstellt werden. Für jedes mit dem FS20-System zu steuernden Gerät wird ein FS20 Zwischenstecker benötigt. Insgesamt ist ein FS20 Sender ausreichend.
