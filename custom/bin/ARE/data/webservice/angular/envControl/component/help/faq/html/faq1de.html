Mit AsTeRICS Ergo können 2 Arten von Geräten gesteuert werden:
  1. Mit AsTeRICS Ergo können die meisten Geräte, die mit einer Infrarot-Fernbedienung bedient werden, gesteuert werden. Hierfür wird ein IrTrans benötigt.
    Beispiele:
    • Fernseher
    • DVD-Player
    • Radio/CD-Player
    • Staubsaugerroboter
    • Jalousien
    • ...
    Wenn Infrarot-Glühbirnen verwendet werden, kann mit AsTeRICS Ergo auch das Raumlicht gesteuert werden.
  2. Außerdem können Geräte gesteuert werden, die an der Steckdose hängen und per Schalter ein- und ausgeschaltet werden. Hierfür wird das FS20-System benötigt.
    Beispiele:
    • Tisch-/Stehlampen
    • Lavalampen
    • Ventilatoren
    • ...
