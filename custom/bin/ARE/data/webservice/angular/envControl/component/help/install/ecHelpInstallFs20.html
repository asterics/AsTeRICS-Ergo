Installation von FS20

Es wird geprüft ob FS20 bereits installiert ist ...

Wenn Sie bereits einen Fs20 besitzen, können Sie diesen wie folgt installieren:

  1. Stecken Sie den FS20-Sender mit dem USB-Kabel an dem Computer an.
  2. Warten Sie 10 Sekunden und überprüfen Sie ob das rote Lämpchen am FS20-Sender leuchtet.
    Leuchtet das Lämpchen nach 10 Sekunden noch?
  3. Es gibt ein Problem mit dem FS20-Sender, das behoben werden muss. Klicken Sie auf "Problem beheben" um das zu tun. Bestätigen Sie die dadurch erscheinende Abfrage mit "Ja".
  4. Stecken Sie den FS20-Sender nochmal vom PC ab und danach wieder an. Das Lämpchen sollte nun durchgehend leuchten.
  5. Klicken Sie auf "Testen" um die korrekte Installation zu prüfen.
    wird getestet... Test fehlgeschlagen Test erfolgreich. Klicken Sie auf Gerät hinzufügen um eine Steuerung mit FS20 einzurichten.
