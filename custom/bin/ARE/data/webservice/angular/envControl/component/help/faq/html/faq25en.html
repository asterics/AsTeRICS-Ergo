The Order-Wizard shows which additional hardware you need to be able to control the devices.

Select all the devices you wish to control by placing a check mark in the box on the left of the device. You can change the number of devices on the right side.

If the device you wish to control is not listed, select “Other remote-controlled device” or “Other socket device”.

Select “Other remote-controlled device” for devices which can be controlled via infrared remote control.

Select “Other socket device” for devices which are plugged into a socket and only switched on and off.
