Installations von IrTrans-USB

Es wird geprüft ob IrTrans bereits installiert ist ...

Wenn Sie bereits einen IrTrans-USB besitzen, können Sie diesen wie folgt installieren:

  1. IrTrans benötigt zusätzliche Software. Bitte klicken Sie auf folgenden Link um diese herunterzuladen: Software IrTrans USB
  2. Installieren Sie die heruntergeladene Software - Folgen Sie den Anweisungen und klicken Sie immer auf "Weiter".
  3. Stecken Sie den IrTrans mit einem USB-Kabel an dem Computer an.
  4. Starten Sie die installierte Software "IRTransServer USB". Klicken Sie dafür auf "Start" und suchen Sie nach "IrTrans":
  5. Klicken Sie auf "Testen" um die korrekte Installation zu prüfen.
    wird getestet... Test fehlgeschlagen Test erfolgreich
IrTrans ist bereits erfolgreich installiert und kann verwendet werden. Klicken sie auf Gerät hinzuzufügen um eine Steuerung mit IrTrans einzurichten.
