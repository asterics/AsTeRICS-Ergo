Installations von IrTrans-USB

Es wird geprüft ob IrTrans bereits installiert ist ...

Wenn Sie bereits einen IrTrans-USB besitzen, können Sie diesen wie folgt installieren:

  1. IrTrans benötigt zusätzliche Software. Bitte klicken Sie auf folgenden Link um diese herunterzuladen:
    Software IrTrans USB
  2. Installieren Sie die heruntergeladene Software - Folgen Sie den Anweisungen und klicken Sie immer auf "Weiter". Bei folgendem Fenster müssen Sie "IrTrans USB" auswählen:

    Wenn Sie möchten, dass IrTrans bei dem Computerstart automatisch startet, dann wählen Sie zusätzlich die Option:
    "Add IRTrans Server to the Startup Folder"
  3. Stecken Sie nach Abschluss der Softwareinstallation den IrTrans mit einem USB-Kabel an dem Computer an.
  4. Warten Sie bis die Hardwareinstallation abgeschlossen ist.
    Klicken Sie auf folgende Meldung bzw. Symbol um den Verlauf der Installation zu sehen:
    Falls die Meldung und das Symbol nicht mehr zu sehen sind, so ist die Hardwareinstallation bereits abgeschlossen.
    Öffnen Sie dieses Fenster um den Verlauf der Installation zu sehen:
    Falls dieses Fenster nicht mehr zu sehen ist, so ist die Hardwareinstallation bereits abgeschlossen.
  5. Starten Sie die installierte Software "IRTransServer USB". Klicken Sie dafür auf "Start" und suchen Sie nach "IrTrans":
  6. Klicken Sie auf "Testen" um die korrekte Installation zu prüfen.
    wird getestet... Test fehlgeschlagen Test erfolgreich. Klicken Sie auf Gerät hinzuzufügen um eine Steuerung mit IrTrans einzurichten.
IrTrans ist bereits erfolgreich installiert und kann verwendet werden. Klicken Sie auf Gerät hinzuzufügen um eine Steuerung mit IrTrans einzurichten.
